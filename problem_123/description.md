## Problem: String Mismatch Counter

### Description
Write a program that reads two strings of the same length and prints the number of positions where the characters differ.

### Input Format
Two lines, each containing a string of the same length

### Output Format
Print a single integer representing the number of mismatches

### Examples

---
Input:
```
hello
hallo
```
Output:
```
1
```

---
Input:
```
abcdef
abcdef
```
Output:
```
0
```

---
Input:
```
ABCDE
abcde
```
Output:
```
5
```

---
Input:
```
programming
prokrammink
```
Output:
```
2
```

---
Input:
```
test
best
```
Output:
```
1
```
