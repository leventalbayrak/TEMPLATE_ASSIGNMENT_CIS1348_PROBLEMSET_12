## Problem: Normalize Spaces

### Description
Write a program that reads a line and normalizes the spaces. Consecutive spaces should be converted to a single space.

### Input Format
Single line containing text with possibly consecutive spaces

### Output Format
Print the line with all consecutive spaces converted to single spaces

### Examples

---
Input:
```
Hello    world  this  is   a    test
```
Output:
```
Hello world this is a test
```

---
Input:
```
one  two    three
```
Output:
```
one two three
```

---
Input:
```
no extra spaces
```
Output:
```
no extra spaces
```

---
Input:
```
a    b    c
```
Output:
```
a b c
```

---
Input:
```
single
```
Output:
```
single
```
