## Problem: Matrix-Vector Multiplication

### Description
Write a program that multiplies a 3Ã—3 matrix by a vector and prints the resulting vector.

### Input Format
Three lines, each containing a matrix row (space-separated) followed by a tab and the corresponding vector element

### Output Format
Print three integers, one per line, representing the resulting vector

### Examples

---
Input:
```
1 0 0	5
0 1 0	3
0 0 1	2
```
Output:
```
5
3
2
```

---
Input:
```
2 0 0	1
0 2 0	2
0 0 2	3
```
Output:
```
2
4
6
```

---
Input:
```
1 2 3	1
4 5 6	0
7 8 9	0
```
Output:
```
1
4
7
```

---
Input:
```
1 0 2	2
0 1 0	3
3 0 1	4
```
Output:
```
10
3
10
```

---
Input:
```
1 1 1	1
2 2 2	1
3 3 3	1
```
Output:
```
3
6
9
```
